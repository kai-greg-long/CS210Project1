# Monopoly Board Simulator
**CS210 – Spring 2026**  
**Kai Long**

---

## Core Features
- Circular linked list Monopoly board  
- Player traversal with dice rolls  
- File-based board initialization  
- Controlled board display from player position  
- Fills board using .txt file containing space data  

---

## Additional Features
- Remove property by name  
- Find properties by color  
- Pass GO tracking  

---

## Instructions
1. Place `monopoly_spaces.txt` in the project directory  
2. Compile the program  
3. Run `main.cpp`  
4. The game will simulate 10 turns of traversal
